Estes códigos a implementar no ESP, quando este está ligado ao Kitsat, permitem:
- criação de um command parser para programação do ESP em C
- troca de informação entre eles
- pedido de dados de sensores do Kitsat

  Neste momento, qualquer função que antes era aplicada na Kitsat Groundstation ou programada com a biblioteca kitsat de Python é possível implementar em C ou C++.
